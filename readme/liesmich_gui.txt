Rasterizer-GUI

Diese Grafische Oberfläche hat die selben einfachen Features, wie die
Kommandozeilenversion. 

Das Kommando

# java -jar rasterizergui.jar

ausführen. Die Oberfläche erscheint (und sollte weitestgehend 
selbserklärend sein).

Die Software wurde unter der GNU GP-Lizenz veröffentlicht. Der Lizenztext
ist in der Datei "Copying.lib" abgelegt, die Teil dieser Veröffentlichung 
ist. Der Quelltext des Programms ist im JAR mit enthalten. Ran da :)

Auf eigene Gefahr verwenden.

Viel Spaß,

Jan
